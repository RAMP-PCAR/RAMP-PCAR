	<details class="margin-bottom-medium">
		<summary class="">[fr] Search Map</summary>
		<div>
			<p>
				[fr] The Data Grid allows you to view a subset of the data that is visible in the
				current map extent.</p>
			<p>
				<b>[fr] Usage:</b></p>
			<p>
				<img src="./App_Themes/images/help/en/OilSands/datagrid1.png" class="image-actual"
					alt="The datagrid columns." /></p>
			<ul>
				<li><span class="bold-text">[fr] Tab Once (Header):</span> You can tab into the first
					row of headers of the grid and use the arrow keys to traverse right or left. By
					pressing spacebar or clicking on a header will sort the data grid by that select
					header in ascending or descending order. </li>
				<li><span class="bold-text">[fr] Tab Twice (Cells):</span> You can traverse through
					the cells of the data grid. </li>
				<li><span class="bold-text">[fr] Tab Thrice (Pagination):</span> You have the option
					of navigating through the pages of the data grid by using the next and previous
					buttons; jumping to the first and last page of the data grid is also possible with
					the buttons. You can also jump to specific page by entering a page number in the
					textbox provided and then pressing enter. </li>
				<li><span class="bold-text">[fr] Zoom To:</span> Every column on the data grid has a
					zoom to option. If the magnifying glass is clicked, the map and the data grid will
					dynamically zoom to that specific point.
					<p>
						<img src="./App_Themes/images/help/en/datagrid2.png" class="image-actual" alt="A cutout of the datagrid with the Zoom button highlighted." /></p>
				</li>
				<li><span class="bold-text">[fr] More Info:</span> The info button on the data grid
					will populate the 'More Info' panel with extended information and metadata.
					<p>
						<img src="./App_Themes/images/help/en/datagrid3.png" class="image-actual" alt="A cutout of the datagrid with a highlight of where to download the data." /></p>
				</li>
			</ul>
		</div>
